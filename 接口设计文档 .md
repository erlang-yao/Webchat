# 接口API文档

本文档描述了聊天系统接口定义
>1.本文档基于 Swagger和Apifox自动生成，仅人为检查修改并增加部分标题序号，以供方便阅读
## 接口分组（Tags）

| Tag | 说明 |
|-----|------|
| `chat-export-controller` | 聊天记录导出模块 |
| `good-friend-controller` | 好友模块：好友列表、最近会话、删除好友 |
| `group-controller` | 群聊模块：群列表、创建群、退出群、搜索群 |
| `group-message-controller` | 群聊消息模块：消息、历史记录 |
| `single-message-controller` | 单聊消息模块：消息、历史记录、已读标记 |
| `sys-controller` | 系统模块：文件上传下载、反馈、敏感词、监控 |
| `Admin Controller` | 系统模块：总的用户管理 |
| `user-controller` | 用户模块：注册、登录、信息管理、好友分组 |
| `validate-message-controller` | 验证消息模块：好友/群聊验证消息 |
